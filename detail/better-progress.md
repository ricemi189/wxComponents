# better-progress

仿 BootStrap 进度条组件

## 使用该组件

1、配置json

```json
{
  "usingComponents": {
    "better-progress": "/components/better-progress/index"
  }
}
```

2、视图层引入

```html
<better-progress class="progress-bar-striped progress-bar-animated progress-bg-info" percentage="{{percentage}}">{{percentage}}%</better-progress>
```
